## 如果DNS 不可用先执行以下命令

```
CONFIG_FILE="/etc/systemd/resolved.conf"
DNS_SETTING="DNS=8.8.8.8"
sudo sed -i "/^\[Resolve\]/a $DNS_SETTING" "$CONFIG_FILE"
sudo systemctl restart systemd-resolved
```
